# Чат-бот с Claude

## Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env`:
   ```
   ANTHROPIC_API_KEY=your_api_key_here
   ```

3. Запустите:
   ```bash
   langgraph dev
   ```

> **Примечание**: Если ваш основной браузер Safari, используйте флаг `--tunel`:
> ```bash
> langgraph dev --tunel
> ```
